
Guia Completo para Publicar sua Aplicação no Firebase

Para que sua aplicação "deo" funcione online e de forma segura, precisamos fazer duas coisas principais:

Configurar um projeto no Firebase com autenticação por e-mail/senha.

Preparar e enviar o código (que agora terá uma tela de login) para o Firebase.

Passo 1: Preparar o Ambiente Local
(Este passo permanece o mesmo)

Instale o Firebase CLI: Se ainda não tiver, abra seu terminal e execute:

npm install -g firebase-tools

Faça login no Firebase: No mesmo terminal, execute o comando:

firebase login

Passo 2: Criar e Configurar o Projeto no Firebase
Acesse o Console do Firebase.

Clique em "Adicionar projeto" e dê um nome a ele, por exemplo, deo-organizador-semanal.

Siga os passos de criação.

Configurar Autenticação por E-mail/Senha:

No menu lateral esquerdo, vá para Build > Authentication.

Clique em "Primeiros passos".

Na aba "Provedores de login", selecione "E-mail/senha" e ative-o. Não ative a opção de link de login por e-mail. Salve.

Adicionar seu Usuário (Acesso exclusivo):

Ainda em Authentication, vá para a aba "Users".

Clique em "Add user".

Digite o seu e-mail e uma senha segura.

Clique em "Add user". Agora, somente você poderá fazer login na aplicação com essas credenciais.

Configurar o Banco de Dados (Firestore):

No menu, vá para Build > Firestore Database.

Clique em "Criar banco de dados" e inicie em modo de produção.

Escolha um local e clique em "Ativar".

Regras de Segurança do Firestore:

Ainda no Firestore, vá para a aba "Regras".

Substitua as regras existentes por estas. Elas garantem que um usuário só pode acessar seus próprios dados.

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /artifacts/{appId}/users/{userId}/{document=**} {
      allow read, write: if request.auth != null && request.auth.uid == userId;
    }
  }
}

Clique em "Publicar".

Passo 3: Conectar seu Código ao Projeto Firebase
Pegue as credenciais do seu app:

Vá para "Configurações do projeto" (ícone de engrenagem).

Na aba "Geral", seção "Seus apps", clique no ícone da web </>.

Dê um apelido (ex: "deo-app") e clique em "Registrar app".

Copie o objeto firebaseConfig que será exibido.

Atualize o código React:

No código da aplicação, você substituirá as variáveis de configuração pelas suas credenciais. O código que fornecerei já estará preparado para isso. Você só precisa colar o objeto firebaseConfig no local indicado.

Passo 4: Publicar a Aplicação (Deploy)
(Este passo permanece o mesmo)

Navegue até a pasta do seu projeto no terminal.

Instale as dependências: npm install

Inicialize o Firebase no seu projeto: firebase init

Selecione Hosting.

Selecione Use an existing project e escolha seu projeto.

public directory? > build

Configure as a single-page app? > y (sim)

Set up automatic builds with GitHub? > n (não)

Construa a aplicação: npm run build

Faça o deploy: firebase deploy

Ao final, você receberá a URL da sua aplicação. Ao acessá-la, você verá uma tela de login antes de poder gerenciar suas tarefas.
